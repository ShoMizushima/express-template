# express-template

Template for backend repository with express.

## Usage(Run)

```
docker-compose up -d
```

## Usage(Test)

```
cd express
npm i
npm test
```

## Library initialize

1. express

   ```
   npm i express
   npm i -D typescript
   npm i -D ts-node
   npm i -D @types/express
   ```

1. eslint

   ```
   npm i -D eslint
   npx eslint --init
   ```

1. prettier

   ```
   npm i -D prettier eslint-config-prettier
   vi .prettierrc
   ```
